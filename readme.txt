This plugin helps to display simple floating social media share buttons.
 
<strong>Features</strong>
* Easy to install
* Simple to use
* Show or Hide Buttons

== Installation ==
1. Upload WS Sharebar to the  /wp-content/plugins/ directory
2. Activate the WS Sharebar plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > WS Sharebar
4. Set Options and Save
